# 家庭语音助手

两套方案、两种思路，都跑通了。

## 项目

| 目录 | 方案 | 一句话 |
|------|------|--------|
| [`voice-doubao/`](voice-doubao/) | 火山引擎端到端实时语音 | 体验好，延迟低，方言，但 LLM 不可控 |
| [`voice-hermes/`](voice-hermes/) | 全开源组件拼装 | LLM 可控（接自己的 Agent），可扩展，但延迟高 |

## 对比

| | voice-doubao | voice-hermes |
|--|--|--|
| 延迟 | <1 秒 | 4-8 秒 |
| 识别准确率 | 95%（懂方言） | 70-80% |
| 唤醒 | 无（VAD 直接触发） | 唤醒词 |
| 打断 | 支持 | 不支持 |
| LLM | 火山模型（黑盒） | 自选（Hermes/DeepSeek/Ollama） |
| 记忆 | 20 轮上下文 | 可接持久记忆 Agent |
| 成本 | 有免费额度，超出付费 | 全免费 |
| 隐私 | 音频上云 | 基本内网 |

## 硬件需求

- Linux 服务器（CPU 即可，无 GPU）
- USB 麦克风
- 3.5mm 有线音响
