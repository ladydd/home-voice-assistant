# 家庭语音助手

两套语音方案 + 一个 ESP32 硬件项目，都跑通了（ESP32 还在规划中）。

## 项目结构

```
├── voice-doubao/     火山引擎端到端实时语音（已部署，永久运行）
├── voice-hermes/     全开源方案，对接 Hermes Agent（可扩展）
├── esp32/            ESP32-S3 无线麦克风终端（规划中）
└── blog/             搭建过程踩坑博客
```

## 两套语音方案对比

| | voice-doubao | voice-hermes |
|--|--|--|
| 延迟 | <1 秒 | 4-8 秒 |
| 识别准确率 | 95%（懂方言） | 70-80% |
| 唤醒 | 无（VAD 直接触发） | 唤醒词 |
| 打断 | 支持 | 不支持 |
| LLM | 火山模型（黑盒） | 自选（Hermes/DeepSeek/Ollama） |
| 记忆 | 20 轮上下文 | 可接持久记忆 Agent |
| 成本 | 有免费额度，超出付费 | 全免费 |
| 隐私 | 音频上云 | 基本内网 |

## 硬件需求

- Linux 服务器（CPU 即可，无 GPU）
- USB 麦克风
- 3.5mm 有线音响
